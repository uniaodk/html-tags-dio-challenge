What is HTML? back

HTML is the standard markup language for creating Web pages. It is NOT a programing language, you could compare HTML as a CSV or XML file.

  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Tags

<h1> to <h6> back

Used to define HTML headings

  • <h1> defines the most important heading. Use only one for the whole page
  • <h6> defines the least important heading

Example:

This is heading 1

This is heading 2

This is heading 3

This is heading 4

This is heading 5
This is heading 6

<p> back

Defines a paragraph.

  • Browsers automatically add a single blank line before and after each <p> element

Example:

This is paragraph 1

This is paragraph 2

This is paragraph 3

<mark> back

Defines text that should be marked or highlighted.

Example:

This is paragraph with text highlighted by tag mark

<small> back

Defines smaller text

  • Small tag shrink the font proportional

Example:

This is paragraph with small text by triple tag small

<i> back

Defines a part of text in an alternate voice or mood.

  • Typically displayed in italic
  • Used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.

Example:

This is paragraph with my apresentation on portuguese, Meu nome é André by tag i

<u> back

Defines some text that is unarticulated and styled differently from normal text.

  • Typically displayed with an underline
  • Used to misspelled words or proper names in Chinese text

Example:

This is some mispeled text.

<strong> back

Defines important text

  • Typically displayed in bold

Example:

Where is more important? This text is important!

<ol> back

Defines an ordered list

  • An ordered list can be numerical or alphabetical

Attributes

Atribute Value Description
reversed reversed Reverse order (9, 8, 7...)
start number Start value to order
type 1, A, a, I, i Kind of marker to order

Example:

  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk

<ul> back

Defines an ordered list

  • Defines an unordered (bulleted) list

Example:

  • Coffee
  • Tea
  • Milk

<li> back

Defines a list item

Attributes

Atribute Value Description
value number Only to <ol> list. Specifies the start value of a list item.

Example:

  1. Coffee
  2. Tea
  3. Milk
  • Coffee
  • Tea
  • Milk

<a> back

Defines a hyperlink, which is used to link from one page to another

Attributes

Atribute Value Description
donwload filename Specifies that the target will be downloaded when a user clicks on the hyperlink
href URL or id tag Specifies the URL of the page the link goes to
rel alternate, author, bookmark, external, help, license, next, nofollow, noreferrer, noopener, prev, search, tag Specifies the relationship between the current document and the linked documento
target _blank, _parent, _self, _top Specifies where to open the linked document

Example:

Visit Github.com

<hr> back

Defines a thematic change in the content

  • Often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page

Example:

The Main Languages of the Web

HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page, and consists of a series of elements. HTML elements tell the browser how to display the content.

CSS is a language that describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work, because it can control the layout of multiple web pages all at once.

<sub> back

Defines subscripted text

  • Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.
  • Subscript text can be used for chemical formulas, like H2O.

Example:

This text contains subscript text.

<sup> back

Defines superscripted text

  • Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.
  • Superscript text can be used for footnotes, like WWW[1]

Example:

This text contains superscript text.
